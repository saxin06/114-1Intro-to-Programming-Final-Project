# 換裝記帳本

## 專案簡介
為大學生設計的小程式。本專案將日常理財與裝扮抽卡機制結合，透過「記帳獲取點數」與「隨機抽取裝扮」的正向回饋循環，解決使用者在培養記帳習慣初期容易放棄的問題。

## 核心功能
* **帳務紀錄與點數系統**：記帳即可獲得點數，建立行為與獎勵的連結。
* **隨機抽取機制**：消耗點數從道具池（帽子、臉型、服裝）中進行隨機抽取。
* **角色視覺化**：利用字符與Emoji即時組合並顯示角色外觀。
* **背包管理**：完整紀錄已獲得的裝扮。

* **開發環境**：Python 3.13.2
* **依賴套件**：Python 標準函式庫（`random`）。
* **執行指令**：
